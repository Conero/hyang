## 描述

## 命令行程序

- 格式
  - php sr.phar {command} {action}  {args}
- build b  **phar 打包工具**
  - action 参数
    -  *.*   当前模块
    - project_dir 项目目录
  - 可选参数
    - name=phar 名称
    - dist=发布目录
  - --options 选项
    - --no-require-script 	无require脚本
- help ?     **文档说明**
